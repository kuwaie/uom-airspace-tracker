# UOM Airspace Tracker

一个基于Web的民航局空域信息可视化工具，为无人机操作者和航空爱好者提供便捷的空域查询体验。

## ✨ 主要功能

- 🗺️ **集成天地图** - 使用高精度的中国地图服务
- 📍 **实时定位** - 支持GPS定位和位置追踪
- 🚁 **空域可视化** - 显示全国七大区域的空域限制信息
- 🎯 **智能加载** - 根据当前位置自动加载对应空域图层
- 🔐 **Token管理** - 安全的Token存储和验证机制
- 📱 **响应式设计** - 适配各种设备屏幕尺寸

## 🛠️ 技术栈

- HTML5 / CSS3 / JavaScript (ES6+)
- Leaflet.js (地图渲染)
- 天地图API (底图服务)
- 民航局UOM空域数据接口

## 🚀 快速开始

1. 获取天地图Token和UOM空域Token【当前实测chrome浏览器可用，其它浏览器未知】
2. 使用浏览器打开 `uom动态定位-全国追踪.html` 即可使用。
3. 输入Token，开始探索空域信息
4. 支持手机端打开，使用手机打开时建议将上述html文件分享到手机中，在文件浏览器中长按，选择使用浏览器打开，并选择系统自带浏览器。

## 📝 使用场景

- 无人机飞行前空域查询
- 飞行路线规划
- 航空教学和演示
- 空域政策研究

## ⚠️ 注意事项

- 需要有效的天地图API Token
- 需要民航局UOM系统的访问权限
- 空域数据仅供参考，实际飞行请遵循官方规定
